# 🏗️ P6 Practice Tracker

> Interactive 90-day Primavera P6 skill-building tracker

A personal, fun, gamified web app to track my journey from PMP-certified Construction PM to job-ready Primavera P6 expert. Built to keep me consistent and motivated through 12 weeks of practice.

## 🎯 What It Tracks

- **12-Week Plan** — Structured weekly focus areas with daily tasks
- **5 Portfolio Projects** — Build increasingly complex schedules with 20-step checklists
- **8 Weekly Challenges** — Build advanced skills (resource leveling, EVM, crashing, etc.)
- **Daily Log** — Log every practice session with hours and notes
- **12 Achievements** — Unlock badges as I progress
- **Streak Counter** — Stay consistent day after day
- **Portfolio Checklist** — Interview-ready assets (PDF, videos, LinkedIn, CV)

## ✨ Features

- 📱 **Mobile-first design** — Works perfectly on phone
- 💾 **Local storage** — All progress saved on device (no account needed)
- 🎉 **Confetti celebrations** when you complete items
- 🏆 **Achievement system** with unlockable badges
- 🔥 **Streak tracking** for daily motivation
- 📊 **Visual progress bars** for all metrics
- 💾 **Backup/restore** — Export your progress as JSON
- 🎨 **Eye-soothing colors** — Sage green, terracotta, gold palette
